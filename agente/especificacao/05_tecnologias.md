# 05 - Tecnologias

## Linguagem e runtime

- Python 3.x

## Interface

- Streamlit

## Dados de mercado e indicadores

- yfinance
- pandas
- pandas-ta-classic

## Testes

- pytest

## Configuracao

- `app/config/settings.py`
- variaveis de ambiente para endpoint/chave/deployments
